Amazon Product Ranking | Bảng Xếp Hạng Sản Phẩm Amazon

Nhập dữ liệu sản phẩm từ Amazon để phân tích. / Enter product data from Amazon for analysis.

📌 Hướng Dẫn Sử Dụng AmazonTrend-AI

  • Bước 1: Vào ứng dụng AmazonTrend.
  • Bước 2: Tích hợp extension Productor for Merch by Amazon trên Chrome.
  • Bước 3: Mở Amazon.com, chọn vùng Mỹ bằng cách điền mã zip 20001 hoặc bất kỳ mã zip nào của Mỹ.
  • Bước 4: Tìm kiếm sản phẩm cần phân tích trend, chọn "Newest Arrivals". Ví dụ, search từ " Shirt ", sau đó chọn Newest Arrivals góc phải trên
  • Bước 5: Đợi website load dữ liệu, và extension Productor hiển thị đầy đủ các chỉ số như Rank BSR, ASIN, Sales, Added...
  • Bước 6: Nhấn Ctrl + A để chọn toàn bộ dữ liệu trên Amazon, sau đó Ctrl + C để sao chép.
  • Bước 7: Dán dữ liệu vào ô phân tích trên AmazonTrend-AI của Lelife, sau đó nhấn Phân tích để nhận kết quả.
